How it works

  1. Upload your file. The site generates a random 3‑digit code.
  2. We store the file in Firebase Storage and save the code → file mapping in Firestore.
  3. On any device, enter the code to fetch the download link.

Tip: For production, consider longer codes or sign‑in, and set automatic deletion/expiry.

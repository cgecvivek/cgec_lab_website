How it works

  1. Upload your file. The site generates a random 3-digit code.
  2. We store the file in Supabase Storage (`secondbucket`) and save the code → file mapping in file_codes table.
  3. On any device, enter the code to fetch the download link.

Tip: Use authentication and longer codes for production.

How it works

  1. Upload your file. The site generates a random 3‑digit code.
  2. We store the file in Supabase Storage (public bucket) and save the code → file mapping in Supabase table file_codes.
  3. On any device, enter the code to fetch the download link.

Tip: For production, consider longer codes, authentication, and automatic file expiration.
